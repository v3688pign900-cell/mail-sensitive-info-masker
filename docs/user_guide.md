# User Guide

## Quick Start

1. Open the website or local `index.html`
2. Paste the email content
3. Add custom rules if needed
4. Click `Mask / 遮罩轉換`
5. Review the mapping table
6. Copy the masked result

## Buttons

- `Mask / 遮罩轉換`: run masking
- `Copy Result / 複製結果`: copy masked output
- `Clear / 清除`: clear all current page data
- `Load Sample / 載入範例`: load fake sample text
- `Add Custom Rule / 新增自訂替換`: add one custom replacement row

## Custom Rule Tips

- Use more specific source strings first
- Custom rules override automatic masking because they run first
- Rules are not saved after refresh or close

## Mapping Review

The mapping table shows:

- type
- original value
- masked label

Use it to confirm the replacements before sharing masked text externally.
